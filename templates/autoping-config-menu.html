Foundry to Discord: Auto-Ping Mapper Tool

This is a tool for pinging a Discord user whenever their @username is mentioned in chat. Enter a keyword (case-sensitive) and assign a Discord user ID or Role ID to it. Whenever that keyword is mentioned in Foundry Chat, for example, the keyword foo, when mentioned in chat as @foo, the ping that corresponds to it will replace the keyword. To get the ID, simply Right-Click on the user or role, and Copy ID. Enable Developer Mode for this to work.
